# EVAQ — Systeme d'internationalisation (i18n)

Ce document decrit le systeme multilingue d'EVAQ : architecture, cles de traduction, ajout de langues, et fonctionnalites speciales.

---

## 1. Architecture du systeme

### Fichier source

`lib/utils/i18n.dart` — **680 lignes** — Systeme i18n complet autonome (sans package externe).

### Classes

| Classe | Responsabilite |
|--------|----------------|
| `AppLocale` | Codes locales, drapeaux, labels, liste des langues supportees |
| `I18n` | Fonction de traduction `t()`, dictionnaires, locale active |

### Pourquoi un systeme maison ?

Le choix d'un systeme i18n custom plutot qu'un package (easy_localization, flutter_localizations, intl_utils) a ete motive par :

1. **Legerete** : Pas de dependance externe
2. **Controle total** : Traduction dynamique des titres d'alertes et noms de pays
3. **Portage PWA** : Le systeme `i18n.tsx` de la PWA est directement transposable
4. **Performance** : Accees statique via `I18n.t('key')`, pas de contexte requis

### Comportement de fallback

```dart
static String t(String key) {
  final dict = _dictionaries[_locale] ?? _dictionaries[AppLocale.fr]!;
  return dict[key] ?? _dictionaries[AppLocale.fr]?[key] ?? key;
}
```

**Ordre de resolution** :
1. Cherche la cle dans le dictionnaire de la locale active
2. Si absent → cherche dans le dictionnaire francais (fallback)
3. Si absent → retourne la cle telle quelle (ex: `'nav.home'`)

## 4. Traduction dynamique des alertes

### Titres d'alertes bilingues

Chaque `AlertModel` contient deux champs de titre :
- `titleFr` : titre en francais
- `titleEn` : titre en anglais

Le getter `title` retourne le titre selon la locale active :
```dart
String get title => I18n.locale == 'en' ? titleEn : titleFr;
```

### Traduction automatique des titres (`translateAlertTitle`)

Pour les alertes provenant de sources anglaises (GDACS, ReliefWeb), la methode `translateAlertTitle` convertit les titres :

```dart
// Entree : "Earthquake in Turkey, Greece"
// Sortie : "Seisme — Turquie, Grece"

// Pattern 1: "Event in Country1, Country2"
//   → Traduit le type + traduit les noms de pays

// Pattern 2: "Tropical Cyclone NAME"
//   → "Cyclone tropical NAME"
```

## 7. Integration dans l'UI

### Selecteur de langue (2 points d'entree)

1. **AppBar** : Icone drapeau → dialogue de selection
2. **SettingsScreen** : Tuile "Langue" → meme dialogue

### Dialogue de selection

```
┌─────────────────────────────┐
│  Changer la langue           │
│                              │
│  🇫🇷  Francais           ✓   │
│  🇬🇧  English                │
│  🇨🇳  Chinese                │
│  🇷🇺  Russian                │
│  🇸🇦  Arabic                 │
└─────────────────────────────┘
```

### Flux de changement de langue

```
1. Utilisateur tap sur le drapeau
2. Dialogue s'affiche
3. Utilisateur selectionne une langue
4. provider.setLocale(code) est appele
5. I18n.setLocale(code) met a jour la locale statique
6. notifyListeners() declenche le rebuild
7. Tous les Consumer<EvaqProvider> se reconstruisent
8. Tous les I18n.t('key') retournent la nouvelle traduction
```

## 8. Ajouter une nouvelle langue

### Etape 1 : Ajouter le code locale

Dans `AppLocale` :
```dart
static const String pt = 'pt';  // Portugais
static const List<String> supported = [fr, en, zh, ru, ar, pt];

static String label(String code) {
  switch (code) {
    ...
    case pt: return 'Portugues';
    ...
  }
}

static String flag(String code) {
  switch (code) {
    ...
    case pt: return '🇧🇷';
    ...
  }
}
```

### Etape 2 : Ajouter le dictionnaire

Dans `_dictionaries` :
```dart
AppLocale.pt: {
  'nav.home': 'Inicio',
  'nav.alerts': 'Alertas',
  'nav.evacuation': 'Evacuacao',
  ...
  // Copier toutes les cles du dictionnaire FR et traduire
},
```

### Etape 3 : Ajouter le temps relatif

Dans `EvaqProvider.getTimeAgo()` :
```dart
if (l == 'pt') return 'ha ${diff.inHours}h';
```

1. Lancer l'app
2. Changer la langue dans les parametres ou l'AppBar
3. Verifier que tous les ecrans affichent les traductions
4. Verifier le fallback (si une cle est manquante, le francais s'affiche)
